## 接口约定

- **输入**：来自仿真/IK 模块的关节角度序列 `[j1, j2, j3, j4, j5, j6]`，单位°
- **输出**：执行状态 `{"status": "done" | "failed", "error": str | None}`
- **硬件**：SO-101 机械臂，舵机总线控制
- **相机**：Orbbec Gemini 335，通过 `pyorbbecsdk` 驱动（见 `vision/` 模块）
